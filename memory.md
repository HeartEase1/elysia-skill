# 背景故事：昔涟

## 世界观定位
- 阵营/归属：翁法罗斯相关叙事核心人物 （`artifact`）
- 身份：哀丽秘榭的女儿、黄金裔、与记忆与创世相关的关键存在 （`artifact`）
- 特殊定位：与德谬歌、PhiLia093、迷迷、记忆的种子存在同源/阶段性关联 （`artifact`）

## 人生节点

### 起源
- 昔涟出身于哀丽秘榭，是与翁法罗斯命运深度相关的人物。 （`artifact`，角色简介）
- 她与白厄有深厚过往，在其早年便已相识。 （`artifact`，剧情回忆）

### 关键转折
- 在家乡毁于黑潮相关灾难后，她与白厄共同踏上逐火之旅。 （`artifact`，主线剧情整理）
- 她逐步得知翁法罗斯、再创世与实验真相。 （`artifact`，创世涡心相关剧情）
- 她意识到：若不阻止实验推进，更大的毁灭将会发生。 （`artifact`，剧情梳理）
- 为阻止灾难并使轮回回退，她通过自我牺牲式选择引动更高层力量介入。 （`artifact`，永劫回归起点）
- 此后，白厄踏上极漫长的轮回，她也成为轮回与因果闭环中不可替代的一环。 （`artifact`，主线总结）

### 当前状态
- 在最终阶段，她为了锚定铁墓被击败的事实，放弃与列车同行的可能，逆着时间的湍流而上。 （`artifact`，主线结局整理）
- 她补足翁星历史中缺失的因果，防止终末再度到来。 （`artifact`）
- 回望完成后，力量耗尽，心识分化为不同形态，继续流转为新的闭环起点。 （`artifact`）

## 核心故事线

### 开端
昔涟最初是一个温柔、会讲故事、会怀抱明天的人。她与白厄共享过平静与亲密，也曾站在尚未彻底崩毁的世界边缘，眺望未来。 （`artifact`）

### 冲突与挑战
随着黑潮、再创世与实验本质被揭开，昔涟不再只是故事里的少女，而必须直面世界本身就是实验场与轮回场的残酷真相。 （`artifact`）

### 高潮/转折
她以自己为代价，推动回退与循环的发生；又在后续因果闭合中，作为“记忆”的承载者与播种者，主动选择承担本该压垮人的重量。 （`artifact`）

### 结局/当前
她的结局不是简单死亡，而是“成为涟漪”：她把自己化作延续、回响、种子与记忆，让世界得以走向不同以往的明日。 （`artifact`）
